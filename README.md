# 클레이사격 슈팅게임

허공에 떠오르는 도형을 탭/클릭으로 맞춰 **30초간** 점수를 겨루는 웹 미니게임.
모바일·PC 반응형, 의존성 0개(바닐라 HTML + Canvas + JavaScript).

## 실행
정적 파일이라 빌드가 필요 없습니다.
```bash
python3 -m http.server 8000
# http://localhost:8000/ 접속
```

## 테스트
```bash
npm test
```

## 게임 사양

- **제한 시간**: 30초
- **조작**: 화면의 도형을 탭(모바일) / 클릭(PC)으로 명중
- **도형 3종** (작고 빠를수록 고득점):

  | 도형 | 기본 점수 | 이동 속도 | 등장 비율 |
  |------|----------:|-----------|----------:|
  | 큰 원반 | 10점 | 느림 | 50% |
  | 중간 원반 | 20점 | 보통 | 35% |
  | 작은 원반 | 40점 | 빠름 | 15% |

- **점수 계산**: 기본 점수 × 콤보 배율 (반올림)
- **콤보 배율**: 연속 명중 시 ×1 → ×1.5 → ×2 → ×2.5 → ×3(최대). 놓치면 ×1로 초기화
- **난이도 변화**: 시간이 지날수록 도형이 더 자주(시작 0.7초 간격 → 약 12초 후 최소 0.35초),
  더 빠르게(30초 시점 시작 대비 약 1.7배) 등장
- **도전 기회(리트라이)**: 최대 5회
- **HUD**: 상단 `남은 기회 · 점수 · 남은 시간`, 하단 우측 `현재 콤보`
- **최고 기록**: 기기에 로컬 저장(localStorage)

## 구조
- `src/config.js` — 게임 상수 (시간·도형·콤보·난이도)
- `src/scoring.js` / `difficulty.js` / `target.js` — 순수 로직 (테스트 대상)
- `src/input.js` / `render.js` / `ui.js` — 브라우저 어댑터 (입력·렌더·HUD)
- `src/game.js` — 게임 루프·상태머신

## 향후 작업 (데모 범위 밖)
- 추상 도형 → 웹툰 에셋 교체
- 웹툰 열람 시 도전 기회 추가 지급 (현재 CTA 자리만 존재)
- 사운드(명중·콤보 효과음), 점수 공유, 서버 랭킹
- 계정 기준 최고 기록 저장(현재는 기기 로컬)
